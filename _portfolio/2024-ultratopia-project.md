## Overview

ULTRATOPIA is a collaborative robotic system for improving pedicle screw placement using ultrasound instead of radiation-based imaging.

## Contributions

- Designed dual-arm robotic system (ultrasound + drilling)
- Developed real-time tissue tracking and navigation
- Led system architecture and integration

## Achievements

- 🏆 Winner of KUKA Innovation Award 2025  
- Demonstrated at MEDICA 2025
